# Instainfo
Get Instagram user information in termux and linux

# linux Installation
```bash
git clone https://github.com/cyberxaman/Instainfo.git
cd Instainfo
sudo apt install python3
bash setup.sh
```

# Termux Installation
```bash
git clone https://github.com/cyberxaman/Instainfo.git
cd Instainfo
apt install python3
bash setup.sh
```

# Usage
```bash
python3 instainfo.py
```
